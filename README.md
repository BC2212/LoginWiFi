# LoginWiFi
Yêu cầu:
    1) Python 3.8 trở lên
    2) AIOHTTP
    3) AIOHTTP_CORS
    4) RouterOS_api

Cài đặt:
    1) Mở Terminal, gõ câu lệnh pip install -r requirements.txt để cài các package
    2) Khởi tạo file config.py tạo thư mục gốc
        "ROUTER = 'ip'
        USERNAME = 'username'
        PASSWORD = 'password'
        "
